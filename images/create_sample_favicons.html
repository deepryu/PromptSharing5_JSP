샘플 파비콘 생성기

이 파일을 브라우저에서 열고 개발자 도구의 콘솔에서 아래 함수를 실행하세요:

📋 설치 방법

  1. 위의 버튼 클릭하여 샘플 파비콘 생성
  2. 다운로드 링크 클릭하여 각 파일 저장
  3. 파일 경로에 복사:
    • favicon-16x16.png → C:\tomcat9\webapps\PromptSharing5_JSP\images\
    • favicon-32x32.png → C:\tomcat9\webapps\PromptSharing5_JSP\images\
    • apple-touch-icon.png → C:\tomcat9\webapps\PromptSharing5_JSP\images\
  4. ICO 파일 생성: 온라인 변환기로 PNG를 ICO로 변환하여 favicon.ico 생성
  5. 브라우저 캐시 삭제 후 확인

🎨 실제 회사 로고 사용

테스트 완료 후, 실제 회사 로고를 사용하려면:

  • favicon.io 또는 realfavicongenerator.net 사용
  • 회사 로고 이미지 업로드
  • 자동 생성된 파일들로 교체
